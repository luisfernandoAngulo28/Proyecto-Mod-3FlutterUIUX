# Screenshots

Esta carpeta contiene las capturas de pantalla de la aplicación.

## Imágenes requeridas:

1. **app-main.png** - Pantalla principal con lista de tareas y estadísticas
2. **app-search.png** - Vista de la búsqueda en acción
3. **app-edit.png** - Modal de creación/edición de tareas

## Instrucciones:

1. Toma screenshots de tu aplicación desde el emulador
2. Guarda las imágenes en esta carpeta con los nombres indicados
3. Formato recomendado: PNG
4. Resolución recomendada: Original del emulador (no es necesario redimensionar)

## Cómo tomar screenshots:

- **Android Studio**: Usa el icono de cámara en la barra lateral del emulador
- **Emulador Android**: Presiona `Ctrl + S` (Windows) o `Cmd + S` (Mac)
- Los screenshots se guardan automáticamente en tu escritorio

## Después de agregar las imágenes:

```bash
git add screenshots/
git commit -m "docs: Agregar capturas de pantalla de la aplicación"
git push
```
